## Introduction
<p>A program that simulates a simple application for managing a bank account. Implementing an initial header with some customer data such as: name, account type, and balance.</p>

<p>Next, a menu describing the operations, such as: check balance, transfer value, receive value, and exit. The menu should appear continuously until the user enters the option to exit.</p>

<p>If they enter any incorrect option, it should display the message "invalid option".</p>

<p>We will use the Scanner class to read the user's option.</p>

## Technologies used
* Java
* Spring Boot

## Tools
The <a href="https://www.oracle.com/br/java/technologies/downloads/" target="_blank"> Java</a> programming language was installed on Windows, version JDK 21.

The program IDE <a href="https://code.visualstudio.com/download" target="_blank"> Spring Tools for Eclipse</a> was installed, using the version 4.31.0.

## Running the project

1. **To get the repository, clone it:**
	```bash
	git clone https://github.com/dornascarol/bank-statement.git
	cd bank-statement
	```

2. **Click the "Run As" button in the Java Application to run the program:**
	```
	Run As > Java Application
	```
